# Literalura 📚

Proyecto desarrollado como parte del Challenge de Literalura del programa de formación de Alura Latam.

## Descripción

Aplicación en Java que permite buscar libros utilizando la API de Gutendex y mostrar información sobre libros y autores a través de un menú interactivo en consola.

## Funcionalidades

- Buscar libro por título
- Listar libros registrados
- Listar autores
- Listar autores vivos en un determinado año
- Listar libros por idioma

## Tecnologías utilizadas

- Java
- IntelliJ IDEA
- Consumo de API (Gutendex)
